# 설치

Context Action 라이브러리를 설치하는 방법을 안내합니다.

## npm으로 설치

```bash
npm install @context-action/core
```

## pnpm으로 설치

```bash
pnpm add @context-action/core
```

## yarn으로 설치

```bash
yarn add @context-action/core
```

## 추가 패키지

### React 통합

```bash
npm install @context-action/react
```

### Jotai 통합

```bash
npm install @context-action/jotai
```
