# 패키지 개요

Context Action은 여러 패키지로 구성되어 있습니다.

## 핵심 패키지

### @context-action/core
- 기본 액션 시스템
- ActionRegister 클래스
- 타입 정의

### @context-action/react
- React 통합
- 훅과 컨텍스트 제공
- 컴포넌트 바인딩

### @context-action/jotai
- Jotai 상태 관리 통합
- 아톰 기반 액션
- 반응형 상태 관리

## 아키텍처

각 패키지는 독립적으로 사용할 수 있으며, 필요에 따라 조합하여 사용할 수 있습니다.
