# @context-action/core

Context Action의 핵심 패키지입니다.

## 개요

이 패키지는 액션 시스템의 기본 기능을 제공합니다.

## 주요 기능

- ActionRegister 클래스
- 타입 정의
- 유틸리티 함수

## 설치

```bash
npm install @context-action/core
```

## 기본 사용법

```typescript
import { ActionRegister } from '@context-action/core'

const register = new ActionRegister()
```
