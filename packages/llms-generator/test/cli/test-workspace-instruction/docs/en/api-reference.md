# API Reference

Complete API documentation.

## Core Classes

### Framework

Main framework class.

#### Methods

##### start()

Starts the framework.

**Returns:** `Promise<void>`

##### stop()

Stops the framework.

**Returns:** `Promise<void>`

## Configuration

### FrameworkConfig

Configuration interface.

#### Properties

- `port`: number - Server port
- `debug`: boolean - Debug mode
